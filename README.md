# Number Serpodile WebSite

should be up at http://serpodile.github.io/number-app-website and http://number.serpodile.com or http://chiffre.serpodile.com

# Install and Run

## pre-requisite (grunt)
```
npm install grunt-cli
```

## Install dep once
```
npm install
````

## Run in dev mode with livereload
```
grunt
```

## Build locally
```
grunt build
```

## Deploy to gh-pages
```
grunt deploy
````
